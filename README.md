## 这本书解决什么问题

离婚不是去民政局签个字那么简单。财产怎么分？孩子归谁？对方转移财产怎么办？被家暴怎么保护自己？

这本书从18年离婚诉讼实务经验出发，60节课覆盖离婚全流程：

| 阶段 | 你会遇到的问题 |
|------|-------------|
| **决定离婚前** | 该不该离？什么时候离？需要做哪些准备？ |
| **起诉离婚** | 怎么选律师？起诉状怎么写？11个问题问律师 |
| **财产分割** | 房产怎么分？对方转移财产怎么查？婚前财产怎么保护？ |
| **抚养权争夺** | 怎么收集有利证据？抚养费要多少合适？探望权怎么约定？ |
| **出轨与家暴** | 怎么合法取证？人身安全保护令怎么申请？损害赔偿怎么要？ |
| **庭审策略** | 证据怎么提交最有效？开庭怎么陈述？法官提问怎么回答？ |
| **二审与执行** | 不服判决怎么办？对方不履行怎么强制执行？ |

## 为什么是这本书

1. **作者是实战派**：许乃义律师执业18年，处理离婚案件1000+，不是纸上谈兵
2. **60节课系统覆盖**：从决定离婚到执行完毕，每一步都有操作指南
3. **2024最新版**：涵盖民法典实施后的最新司法实践
4. **诉讼策略思维**：不仅告诉你怎么做，更教你怎么想——理解法官的裁判逻辑
